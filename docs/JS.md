### [Built-in Values: Infinity]
*_symbol_: `infinity`
*_pronunciation: "infinity", "infinity value"
*_examples_:
```javascript

Infinity

```

### [Built-in Values: undefined]
*_symbol_: `undefined`
*_pronunciation: "variable has not been assigned a value"
*_examples_:
```javascript
Test if a variable is undefined:

var x;

if (x === undefined) {
    txt = "x is undefined";
} else {
    txt = "x is defined";
}
The result of txt will be:

x is undefined
```
