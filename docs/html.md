### 1. `<ul>` -- Unordered List

* *type:* block-level
* *content:* flow content
* *support:* 
* *example:*
```
<ul>
  <li>first item</li>
  <li>second item</li>
  <li>third item</li>
</ul>
```

### 2. `<ol>` -- Ordered List

* *type:* block-level
* *content:* flow content/palpable content.
* *support:* Chrome, Firefox, Safari
* *example:*
```
<ol>
  <li>first item</li>
  <li>second item</li>
  <li>third item</li>
</ol>
```
